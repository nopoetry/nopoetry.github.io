Maven（mvn）是基于项目对象模型(POM project object model)，可以通过一小段描述信息（配置）来管理项目的构建，报告和文档的软件项目管理工具（百度百科）

简单理解为一个打包工具，其他的打包工具比如gradle也很火，常用于与jenkins等代码发布平台的联合代码发布

本文主要介绍linux下的maven的安装方法
